### 출력 

 <p>두 원소를 정확히 한 번 교환하여 LIS의 길이가 감소하지 않도록 만들 수 있다면 첫째 줄에 <span style="color:#e74c3c;"><code>YES</code></span>를 출력한다. 이 경우 둘째 줄에 교환할 두 원소의 인덱스 <mjx-container class="MathJax" jax="CHTML" style="font-size: 99.9%; position: relative;"><mjx-math class="MJX-TEX" aria-hidden="true"><mjx-mi class="mjx-i"><mjx-c class="mjx-c1D456 TEX-I"></mjx-c></mjx-mi></mjx-math><mjx-assistive-mml unselectable="on" display="inline"><math xmlns="http://www.w3.org/1998/Math/MathML"><mi>i</mi></math></mjx-assistive-mml><span aria-hidden="true" class="no-mathjax mjx-copytext">$i$</span></mjx-container>, <mjx-container class="MathJax" jax="CHTML" style="font-size: 99.9%; position: relative;"><mjx-math class="MJX-TEX" aria-hidden="true"><mjx-mi class="mjx-i"><mjx-c class="mjx-c1D457 TEX-I"></mjx-c></mjx-mi></mjx-math><mjx-assistive-mml unselectable="on" display="inline"><math xmlns="http://www.w3.org/1998/Math/MathML"><mi>j</mi></math></mjx-assistive-mml><span aria-hidden="true" class="no-mathjax mjx-copytext">$j$</span></mjx-container>를 공백으로 구분하여 오름차순으로 출력한다. <p>그러한 교환이 존재하지 않으면 첫째 줄에 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</p>
